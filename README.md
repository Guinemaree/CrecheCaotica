# CrecheCaotica
olá pequeno gafanhoto, segue a Lore do projeto.
Você é um funcionário de uma creche com 10 crianças mutantes
objetivo do jogo é salvar todas as crianças de uma invasão de Hides e Queimados (monstros), se deixar alguma criança morrer, você perde.
Classe pessoa: Id, nome e poder, xp.
criança 0 - Hello Kitty / poder de cura
criança 1 - Kuromi / Invisibilidade
criança 2 - Ladybug / Poder da criação
criança 3 - Cat Noir / Poder da destruição
criança 4 - Homem-aranha / Solta teia de aranha
criança 5 - Batman / Sabe voar
criança 6 - Coringa / Telepatia
criança 7 - Arlequina / Telecinese
criança 8 - Chaves / Super audição
criança 9 - Chiquinha / Avatar da Terra
